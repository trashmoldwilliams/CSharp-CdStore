# CD Store

#### By Will Johnson

## Description

An early C# assignment for allowing the user to create CD listings with various properties. Simply it is just object practice in the new language at the time.

## Setup/Installation Requirements

* Windows Machine
* Powershell

* Clone Project
* Run DNU Restore
* Run DNX Kestrel
